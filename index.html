What is F#

F# is a functional-first, universal programming language for writing succinct, robust and performant code.

What is Fable

Fable is a compiler that brings F# into the JavaScript* ecosystem

*As of version 4 Fable can also generate Python, Rust, Dart, PHP and TypeScript.

Functional style

  • Functions are first class citizens
  • Functions are deterministic
  • Data is immutable

                        // Define a list of names
                        let names = [ "Peter"; "Julia"; "Xi" ]

                        // Define a function that takes a name and produces a greeting.
                        let getGreeting name = $"Hello, {name}"

                        // Print a greeting for each name!
                        names
                        |> List.map getGreeting
                        |> List.iter (fun greeting ->
                            printfn $"{greeting}! Enjoy your life.")

                        // Prints:
                        // Hello, Peter! Enjoy your life.
                        // Hello, Julia! Enjoy your life.
                        // Hello, Xi! Enjoy your life.
                    

                        // Define the square function
                        let square x = x * x

                        // Define the sumOfSquares function
                        let sumOfSquares n =
                           [1..n]
                           |> List.map square
                           |> List.sum

                        // Calculate sum of squares of numbers from 1 to 100
                        sumOfSquares 100

                        // Prints: 338350

                        type Face =
                            | Ace | King | Queen | Jack
                            | Number of int

                        type Color =
                            | Spades | Hearts | Diamonds | Clubs

                        type Card = (Face * Color)

                        let aceOfHearts = (Ace, Hearts)
                        let tenOfSpades = (Number 10, Hearts)

                        match tenOfSpades with
                        | Ace, Hearts -> printfn "Ace Of Hearts!"
                        | (Number 10), Spades -> printfn "10 of Spades"
                        | _, Hearts -> printfn "A lovely heart"
                        | _, (Diamonds|Clubs) -> printfn "Diamonds or clubs"

Why does this matter?

Functional style makes programs easier to compose, understand and reason about.

Solid type system makes modeling and refactoring (almost) a joyful experience.

Both provide an understanding of things to come.
